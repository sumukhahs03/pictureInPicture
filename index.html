You can share any of your open windows as a pop out Picture-in-Picture overlay screen.

The following steps shall guide you on how to use this site:

  • The Start button is used to begin the screen share. It prompts you to select what you want to see in the picture-in-picture window.
  • The pop out button can then be clicked after the screen has been shared to get a pop out window that displays the required screen which is overlaid on all other applications. The size of it can also be customized!
  • The stop button is used to stop picture-in-picture mode.
  • Remember, the pop-out button is not active until the start button has been pressed.
